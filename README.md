# 🪵 Carrom Game

Carrom Game is an interactive web-based game built using HTML, CSS, and JavaScript.  
It simulates a digital version of the classic carrom board game with smooth gameplay and engaging UI.

---

## 🚀 Live Demo
🔗 https://ps-carrom-game.netlify.app/

---

## ✨ Features
- 🎯 Interactive carrom gameplay  
- 🕹️ Smooth controls and movements  
- 🎨 Clean and attractive UI  
- ⚡ Real-time game interactions  
- 📱 Responsive design  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ How It Works
- Player controls the striker to hit coins  
- Physics-based movement using JavaScript  
- Real-time updates for smooth gameplay
